Hash Test By Mauricio Grejanin
------------------------------

## Como instalar e executar:

```
# Clonar o repo
$ git clone https://github.com/mgrejanin/hash-test.git

# Navegar até a pasta do projeto
cd ./hash-test

# Instalar as dependências.
$ npm install

# Iniciar aplicação:
npm start

# Build:
npm run build

# Executar testes unitários:
npm run test
```
